Text manipulation

Bold Text

This text is surrounded by <b>

It increases the font thickness to make it more visible. However, it does not give the browser information about the greater importance of this text

(Devdocs Documentation Link)
This is text is surrounded by <strong>

This tag gives the text a certain importance and should be preferred over <b>

(Devdocs Documentation Link)

Italicized Text

This text is italicized by <i>

This tag represents text that is set off from the normal prose, such a foreign word, fictional character thoughts, or when the text refers to the definition of a word instead of representing its semantic meaning.

Example:
"The Queen Mary sailed last night". Here, there is no added emphasis or importance on the word "Queen Mary". It is merely indicated that the object in question is not a queen named Mary, but a ship named Queen Mary.

(Devdocs Documentation Link)
This text is italicized by <em>

This tag represents stress emphasis of its contents and should be used instead of <i>tag

(Devdocs Documentation Link)
This text is italicized by <cite>

The title of a work, such as the name of a book or movie, should use this tag.

(Devdocs Documentation Link)

Site Formatting

<br> is used to make line breaks in your text or between other elements. It is a self-closing tag, which means the tag doesn't need to be closed with a </br> at the end. It closes automatically.

(Devdocs Documentation Link)
<hr> is used to make horizontal lines between other elements. It is a self-closing tag, which means the tag doesn't need to be closed with a </hr> at the end. It closes automatically.

(Devdocs Documentation Link)
